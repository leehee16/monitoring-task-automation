# 자동화 스크래핑 프로젝트

이 프로젝트는 웹 스크래핑과 이미지 분류를 자동화하고, 나이로 미성년자를 분류하기 위한 도구입니다.

## 프로젝트 구조
민감한 정보때문에 몇가지 파일은 업로드하지 않았습니다.
```
.
├── scraper/           # 웹 스크래핑 관련 코드
│   ├── scraper.js     # 메인 스크래핑 로직
│   ├── login.js       # 로그인 처리
│   ├── config.js      # 설정 파일
│   └── ...
│
├── sorter/            # 이미지 분류 관련 코드
│   ├── main.py        # 메인 실행 파일
│   ├── image_classifier.py  # 이미지 분류 로직
│   └── utils.py       # 유틸리티 함수
│
└── prediction/        # 예측 모델 관련 코드
    ├── src/           # 소스 코드
    ├── config/        # 설정 파일
    ├── data/          # 데이터 저장소
    ├── output/        # 출력 결과물
    ├── notebooks/     # 주피터 노트북
    └── main.py        # 메인 실행 파일
```

## 기능

- 웹 스크래핑 자동화
- 이미지 분류 및 처리
- 리포트 생성
- 데이터 예측 및 분석
- 미성년자 분류
